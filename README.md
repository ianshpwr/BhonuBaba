# BhonuBaba 

A visually stunning, full-stack e-commerce platform built with a modern peach pastel aesthetic.

## Tech Stack
- **Frontend**: Next.js App Router, Tailwind CSS, TypeScript
- **Backend**: Node.js, Express, MongoDB Atlas, Mongoose
- **Auth**: JWT Authentication

## Project Structure
- `/client` - Next.js frontend application
- `/server` - Express backend API

## Setup Instructions

### 1. Backend Setup
1. Navigate to the server folder: `cd server`
2. Install dependencies: `npm install`
3. Start the developmental server: `npm run dev` (Runs on `http://localhost:5000`)

*Note: A `.env` file is already pre-configured to point to the dedicated MongoDB Atlas instance.*

### 2. Frontend Setup
1. Open a new terminal and navigate to the client folder: `cd client`
2. Install dependencies: `npm install` (and `npm install lucide-react` if not already installed)
3. Start the Next.js app: `npm run dev` (Runs on `http://localhost:3000`)

Enjoy the aesthetic shopping experience! 🍑
